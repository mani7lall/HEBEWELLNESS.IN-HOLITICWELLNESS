# Sub-Orchestration
This directory contains the core Next.js application for Hebe Wellness.

## 🛠 Active Tools
- **Next.js 14 (App Router)**
- **Tailwind CSS**
- **Framer Motion**

## ⚡️ Component Strategy
- Use `glass-card` for interactive elements.
- Maintain `brand-neon` (#00F59B) as the primary action color.
- All biological data must be formatted in `font-mono` for clinical precision.

- **Tool Logic**: Use `ultrabrain` (GPT-4o / GPT-5 class).
- **Content/Blog**: Use `deep` (NVIDIA Llama 3.1 405B).
